1 About

1.1 About

The purpose of this library is to provide an interface between the Cplint1 suite for SWI Prolog2 and R3 to handle graphs made with the ggplot24 package.

This means that most C3js5 graphing functions of Cplint are also available for R.

1.2 Terminology

Any code between < > represents pseudocode.

The symbol # represents an unsigned integer.

Optional parameters are encapsuled between [ ].
